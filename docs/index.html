How discovery works: Every Monday the agent searches Google (via Serper.dev) and scrapes Instagram hashtags like #babysleepsingapore and #sleeptraining to find candidate accounts. Follower counts and grades come from a manually-updated benchmark table (Social Blade blocks automated scraping), so accuracy depends on when it was last refreshed below.
How to update follower benchmarks (~5 min, once a month):
  1. Visit each competitor's Instagram profile and note their current follower count.
  2. Go to GitHub → your repo → scrapers/competitor_discovery.py → click the pencil (Edit) icon.
  3. Find KNOWN_BENCHMARKS and update each follower number.
  4. Update BENCHMARKS_LAST_UPDATED to today's date (format: "YYYY-MM-DD").
  5. Commit, then re-run the agent so the dashboard picks up the new data and date.
